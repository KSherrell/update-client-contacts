# Update Client Contacts example
## Description
This is a Google Sheets application that reads from a list of company names and email addresses, then creates a Contact group for each unique company name and adds the corresponding email addresses to those groups.


## Features
- One-click to run the app
- Adding, removing, and updating Contacts groups can all happen with that click
